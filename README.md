## 📌 Overview

This project focuses on the **statistical validation** of the **Composite Surgical Metrics Score (CSMS)** — a multi-dimensional scoring system designed to evaluate and benchmark surgical performance. By combining multiple surgical indicators into a single composite score, CSMS aims to provide a more holistic view of surgical quality than any single metric alone.

---

## 🎯 Objectives

- ✅ Validate the reliability and consistency of the Composite Surgical Metrics Score
- 📊 Perform statistical analysis on real-world surgical data
- 🔍 Identify key predictors that drive composite scores
- 📈 Visualize patterns and trends across surgical metrics
- 📝 Generate a comprehensive research report with findings

## 📊 Key Analyses Performed

- 🧪 **Reliability Testing** — Cronbach's Alpha, Inter-rater reliability
- 📐 **Construct Validity** — Confirmatory & Exploratory Factor Analysis
- 📈 **Predictive Validity** — Regression modeling on surgical outcomes
- 📉 **Descriptive Statistics** — Distribution analysis of all metric components
- 🔗 **Correlation Analysis** — Identifying relationships between sub-scores

## 🚀 Getting Started

### Prerequisites

```bash
pip install pandas numpy matplotlib seaborn scipy statsmodels jupyter
```

### Run the Analysis

```bash
# Clone the repository
git clone https://github.com/dhruvjain1999/Validation-of-the-Composite-Surgical-Metrics-Score.git

# Navigate into the directory
cd Validation-of-the-Composite-Surgical-Metrics-Score

# Launch Jupyter Notebook
jupyter notebook
```

Then open any notebook inside the `code/` folder to explore the analysis step by step.
