## 📌 Overview

This project focuses on the **statistical validation** of the **Composite Surgical Metrics Score (CSMS)** — a multi-dimensional scoring system designed to evaluate and benchmark surgical performance. By combining multiple surgical indicators into a single composite score, CSMS aims to provide a more holistic view of surgical quality than any single metric alone.

---

## 🎯 Objectives

- ✅ Validate the reliability and consistency of the Composite Surgical Metrics Score
- 📊 Perform statistical analysis on real-world surgical data
- 🔍 Identify key predictors that drive composite scores
- 📈 Visualize patterns and trends across surgical metrics
- 📝 Generate a comprehensive research report with findings
